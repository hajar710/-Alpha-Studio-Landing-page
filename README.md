# 💎 Alpha Studio - Digital Excellence Landing Page

A sleek, professional, and high-performance landing page for a digital agency. This project focuses on **Strong Fundamentals**, clean typography, and precise spacing to deliver a premium user experience.


---

## ✨ Key Features
* **Sticky Navigation:** A modern dark-themed header that stays visible while scrolling for better UX.
* **Hero Section with Overlay:** High-impact visual using background images with a professional dark overlay for text readability.
* **Showcase Grid:** A curated gallery of featured projects (Fintech, Dashboard, Corporate, etc.) styled with clean cards and hover-ready links.
* **Functional Contact Section:** Includes business details and a stylized contact form with custom input field designs.
* **Pure Layout Management:** Built using **Inline-block** and **Flexbox** strategies to ensure structural integrity without relying on CSS Grid.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom styles featuring advanced selectors, pseudo-elements (`::before`), and positioning.
* **Design Focus:** Strong emphasis on **Typography** (Inter, Segoe UI) and **Spacing**.
